# Python基础

## 常用的数据结构 
- List 
- Tuple-小括号不可改 
- Set-花括号不重复，自动排序 
- Dic-键值对，键必须使用不可变类型，也就是说list和包含可变类型的Tuple不能做关键字/Key；同一个字典中键不能重复【注：是否是可变类型可以用哈希来判断】 

## 变量的数据类型 
- Number 
- String 
- List 
- Tuple 
- Dic

## 一些常用方法

- =  赋值
